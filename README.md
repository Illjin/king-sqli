
# King Sqli

King is a vulnerable site finder, using google dorking. an Python Based tool.





## Deployment

To deploy this project run

```bash
  python3 king.py -h
  python3 king.py --dorks ##Dork Type --pages ## pages you want


```



## Features

- Easy to Use
- Cross Plataform
- An Lite Tool
